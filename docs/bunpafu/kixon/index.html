主題

bipuuk は主題優勢言語です。

自然数 意味
5 主題
150 二人称
174 美しい 
( ( [150 あなた] ) [5 題]
) ( [174 美しい] ) [14 述]

あなたは美しい。

主題は普通左側に来ます。

日本語の「～は」には限定などの意味があるのに対して、bipuuk の 5 にまだそのような意味が設定されていないことに注意してください。
