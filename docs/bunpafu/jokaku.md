## 嗜好、感情

以下の２つは同じです。

```
( ( ( 559 ) 23 ) 5
) ( ( 265 ) 15
  ) ( 176 ) 14
```
> ```
> ( ( ( 1ST ) SBJ ) TOP
> ) ( ( apple ) OBJ
>   ) ( like ) VT
> ```
> 私（が）はリンゴを好む。


```

) ( ( 265 ) 23

```

> ( ( ( 1ST ) DAT ) TOP
